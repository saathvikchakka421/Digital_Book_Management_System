# Digital Book Management System

A simple Python-based digital book management system that helps manage book records, student details, and issue/return operations using a command-line interface.

This project was built to practice core software development concepts like modular code design, CRUD operations, and database integration.

---

## Features

- Add and view books
- Add and view students
- Issue and return books
- Track book availability
- Persistent storage using SQLite
- Modular multi-file structure

---

## Tech Stack

- Python
- SQLite (built-in database)
- VS Code
